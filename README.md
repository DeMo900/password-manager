# purpose of the project

- Creating a strong passwords
- Users can select its length
- And assign it to an app

## *Why to use?*
- All in one place
- *Strong passwords*
- Easy to manage
- Simple Ui
- You can login with google
- extremly smooth and fast *(thanks for redis)*

### specs
- Create an account or enter with your google account 
- Type your app name and the password length you want (minimum 6)
- You get a list with your apps and their password feel free to copy and paste

  
  #### *Used techs*
  - Nodejs
  - Expressjs
  - Redis
  - Mongodb
  - Typescript
  - Tailwind
  - Javascript
  - validation using zod
  - hashing using bcrypt
  - Passport for oauth2.0
  - Nodemailer for sending mails
